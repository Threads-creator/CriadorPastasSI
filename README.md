## CriadorPastasSI
Criador de pastas para o curso Sistemas de Informação IFES. Software utiliza **Java** versao 8 do sdk e **JavaFX 8.0.171**. Cria uma arvore de pastas e subpastas definidos em um arquivo de configuração em Json.
